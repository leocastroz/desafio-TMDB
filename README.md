## Node Version

`v18.17.1`

## Clone

`git@github.com:leocastroz/desafio-vue3.git`

## Installation

`yarn install` or `npm install`

## Starting

`yarn dev` or `npm run dev`

## Compile

`yarn build` or `npm run build`
